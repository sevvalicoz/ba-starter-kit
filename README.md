# ba-starter-kit

Bu repo, iş analizi (Business Analysis) portföyü için örnek doküman ve şablonları içerir:  
- Gereksinim dokümanı (SRS)  
- User story seti  
- Use case’ler  
- Süreç akış diyagramları (process flows)  
- İletişim planı  
- RACI matrisi  
- Gereksinim İzlenebilirlik Matrisi (RTM)
